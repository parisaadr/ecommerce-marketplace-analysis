# ecommerce-marketplace-analysis

## Project Overview

This project analyzes a Brazilian e-commerce marketplace dataset
to understand:

- Customer retention
- Order fulfillment efficiency
- Seller performance
- Product quality

## Business Questions

1. What does the order funnel look like?
2. How frequently do customers return?
3. Which sellers create operational risk?
4. Which products generate the most complaints?
5. How do delivery delays impact customer satisfaction?

## Tools

- SQL (MySQL 8)
- Window Functions
- CTEs
- Cohort Analysis
